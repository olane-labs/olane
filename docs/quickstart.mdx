## Setup your development environment

Learn how to update your docs locally and deploy them to the public.

### Install Olane

<CodeGroup>

```bash npm
npm install @olane/o-protocol @olane/o-config
```

```bash yarn
yarn add @olane/o-protocol @olane/o-config
```

```bash pnpm
pnpm add @olane/o-protocol @olane/o-config
```

</CodeGroup>

### Create your first node

Create a new file called `node.js` and add the following code:

```javascript
import { createNode } from '@olane/o-protocol';
import { defaultConfig } from '@olane/o-config';

async function main() {
  // Create a new Olane node
  const node = await createNode({
    ...defaultConfig,
    // Your node configuration
    name: 'my-first-node',
    port: 3000
  });

  // Start the node
  await node.start();
  
  console.log('Node started successfully!');
  console.log(`Node ID: ${node.id}`);
  console.log(`Listening on port: ${node.port}`);
}

main().catch(console.error);
```

### Run your node

Execute your node with:

```bash
node node.js
```

You should see output similar to:
```
Node started successfully!
Node ID: 12D3KooWH3uVF6wv9Z8...
Listening on port: 3000
```
